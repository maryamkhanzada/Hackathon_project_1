# Physical AI & Humanoid Robotics Documentation

Welcome to the comprehensive guide for building autonomous humanoid robots!

## About This Book

This book provides a practical, hands-on approach to understanding and implementing Physical AI systems with humanoid robotics. You'll learn how to integrate cutting-edge technologies including ROS 2, simulation environments, NVIDIA Isaac, and Vision-Language-Action (VLA) systems.

## Book Structure

### [Introduction](./intro/index.mdx)
Get started with the fundamentals of Physical AI and embodied intelligence.

### [Module 1: ROS 2 Basics](./module1-ros2/index.mdx)
Learn the robotic nervous system - ROS 2 architecture, Python integration, and URDF.

### [Module 2: Digital Twin](./module2-digital-twin/index.mdx)
Master physics simulation with Gazebo and Unity for humanoid robots.

### [Module 3: NVIDIA Isaac](./module3-nvidia-isaac/index.mdx)
Explore AI-powered perception, navigation, and reinforcement learning.

### [Module 4: Vision-Language-Action](./module4-vla/index.mdx)
Integrate LLMs with robotics for cognitive planning and multi-modal interaction.

### [Capstone Project](./capstone-project/index.mdx)
Build a complete autonomous humanoid system integrating all modules.

### [Appendices](./appendices/index.mdx)
Hardware/software requirements, glossary, references, and lab setup tips.

## Learning Path

1. **Beginner**: Start with the Introduction and Module 1 (ROS 2)
2. **Intermediate**: Progress through Modules 2-4 for specialized skills
3. **Advanced**: Complete the Capstone Project integrating everything

## Prerequisites

- Basic programming knowledge (Python preferred)
- Linux fundamentals
- Understanding of robotics concepts (helpful but not required)
