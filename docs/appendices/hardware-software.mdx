# Appendix A: Hardware and Software Requirements

Complete hardware and software specifications for building and deploying autonomous humanoid systems covered in this book.

## Software Requirements

### Operating System

**Required**: Ubuntu 22.04 LTS (Jammy Jellyfish)

```bash
# Check version
lsb_release -a
# Should show: Ubuntu 22.04.x LTS
```

**Why Ubuntu 22.04?**
- Native support for ROS 2 Humble (LTS)
- Long-term support until 2027
- Best compatibility with NVIDIA Isaac Sim and Isaac ROS

### ROS 2 Distribution

**Required**: ROS 2 Humble Hawksbill (LTS)

```bash
# Install
sudo apt install ros-humble-desktop-full

# Verify
ros2 --version
```

**Package dependencies**:
```bash
# Navigation
sudo apt install ros-humble-navigation2 ros-humble-nav2-bringup

# Manipulation
sudo apt install ros-humble-moveit

# Isaac ROS
sudo apt install ros-humble-isaac-ros-visual-slam \
                 ros-humble-isaac-ros-dnn-inference
```

### NVIDIA Isaac Sim

**Version**: 2023.1.1 or later
**Installation**: Via NVIDIA Omniverse Launcher
**Size**: ~50GB
**Requirements**: NVIDIA RTX GPU, 32GB RAM, 100GB storage

### Python Environment

**Version**: Python 3.10+

**Core packages**:
```bash
pip install openai==1.3.0 openai-whisper torch torchvision \
            transformers clip-anytorch ultralytics \
            opencv-python pyaudio gtts mediapipe
```
