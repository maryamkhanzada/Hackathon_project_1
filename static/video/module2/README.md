# Module 2 Videos

This directory contains video demonstrations for Module 2: Digital Twin.

## Required Videos

### Physics Simulation
- `physics_comparison.mp4` - Side-by-side comparison of different physics engines (ODE, Bullet, DART)
- `collision_detection.mp4` - Demonstration of collision detection algorithms
- `timestep_effects.mp4` - Impact of different timestep sizes on simulation stability

### Gazebo Setup
- `gazebo_install.mp4` - Installation walkthrough for Gazebo Garden
- `gazebo_first_launch.mp4` - First-time Gazebo launch and GUI tour
- `gazebo_ros_bridge.mp4` - ROS 2 bridge communication demonstration

### Unity Setup
- `unity_install.mp4` - Unity Hub and Editor installation
- `unity_robotics_hub.mp4` - Installing Unity Robotics Hub packages
- `unity_ros_connection.mp4` - Testing ROS-Unity communication

### Sensors
- `camera_simulation.mp4` - RGB camera with noise demonstration
- `depth_camera_demo.mp4` - Depth camera and point cloud visualization
- `lidar_2d_scan.mp4` - 2D LiDAR scanning animation
- `lidar_3d_pointcloud.mp4` - 3D LiDAR rotating scan
- `imu_data_viz.mp4` - IMU data visualization (orientation, accel, gyro)

### Collisions & Dynamics
- `friction_comparison.mp4` - Different friction coefficients demonstration
- `mass_inertia_effects.mp4` - Impact of mass/inertia on robot motion
- `joint_dynamics.mp4` - Joint damping and friction effects
- `contact_stiffness.mp4` - Contact stiffness tuning (soft vs. stiff)

### Humanoid Lab
- `humanoid_spawn.mp4` - Spawning humanoid in Gazebo
- `sensor_integration.mp4` - All sensors working together
- `balance_test.mp4` - Static balance validation
- `squat_motion.mp4` - Basic joint control demonstration

## Video Specifications

- **Format**: MP4 (H.264 codec)
- **Resolution**: 1920x1080 (1080p) or 1280x720 (720p)
- **Frame Rate**: 30 fps
- **Duration**: 15-60 seconds per video
- **Compression**: Medium quality (balance file size and clarity)

- **Video Editing**: DaVinci Resolve, Kdenlive, OpenShot
- **GIF Conversion**: ffmpeg, ezgif.com
